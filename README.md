# Face Recognition & Attendance System

## 📌 Description
This project automates attendance using face recognition technology.

## 🛠️ Technologies Used
- Python
- OpenCV
- face_recognition
- pandas

## ▶️ How to Run
1. Install Python 3.11
2. Install required libraries
3. Add student images to photos folder
4. Run:
   python Main.py

## 📂 Project Structure
- Main.py
- config.json
- photos/

## 📊 Output
Attendance is stored in Excel format.
